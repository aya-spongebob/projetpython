# Projet Python – Analyse de Corpus et Moteur de Recherche (TD3 → TD7)

Ce projet regroupe les travaux réalisés du TD3 au TD7.  
L’objectif est de construire un mini–moteur de recherche capable :

- de charger un corpus (Reddit + Arxiv)  
- d’analyser les textes  
- d’extraire des statistiques (TF, DF)  
- d’afficher des concordances  
- de rechercher des documents avec un score de similarité (TF-IDF)
